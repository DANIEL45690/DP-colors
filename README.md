## 📖 О проекте

**DP-colors** — это мощное расширение для Visual Studio Code, которое позволяет мгновенно менять цветовую схему редактора с помощью готовых или пользовательских палитр. Более 50+ встроенных цветовых решений для любых задач и настроения.

## 🚀 Установка

### Из маркетплейса VS Code
1. Откройте VS Code
2. Нажмите `Ctrl+Shift+X` / `Cmd+Shift+X`
3. Найдите `DP-colors`
4. Нажмите **Install**

### Через командную строку
```bash
code --install-extension dp-colors
